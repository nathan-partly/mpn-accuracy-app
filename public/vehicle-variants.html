Vehicle Variant Inspector · NZ + UK Sample

What is a variant, exactly?

For every chassis number that returned more than one variant from Partly's vehicle search API, this tool shows you what those variants actually look like — and which properties differ between them.

Built from 1,578 NZ + 1,127 UK chassis numbers. 49 returned multiple variants. Only those are inspected here.

Overall coverage

How to read this

YES / NO
Whether the API recognized the chassis number and returned vehicle data
variant_count
Number of variant records the API returned for one chassis number. Often >1 because a chassis maps to many trims/markets/years
property-distinct
How many of those variants are actually different at the property level (vs being identical bundles with different IDs)
varying axis
A property (grade, market, model_code…) that takes 2+ different values across the variants of one VIN. These are highlighted in orange
dupe
A variant whose properties are identical to another variant of the same VIN. Suggests catalog redundancy, not real ambiguity
assembly comparison
For a sample of VINs, fetched parts catalog (diagrams + parts) per variant from assemblies.v4.search using each variant's ID. Pairs are then compared to see whether property-duplicate variants return identical parts. If yes → genuine catalog redundancy. If no → variant ID encodes hidden differentiation that the property bundle doesn't expose.

Variant selector demo

Live demonstration: given a VIN with N variants returned by upstream, ask the repairer only the questions needed to narrow to one variant. At each step, the selector picks the property that splits the remaining variants most evenly (highest information gain). Equipment lists are split into per-item yes/no toggles since a repairer can answer "does this car have navigation?" but not "pick the right comma-separated combo." Brand explorer

Drill into individual chassis numbers and see exactly which properties differ between the variants returned.
Method: Each chassis was queried via the Partly Retool VIN Sampler proxy. Multi-variant VINs (variant_count > 1) had their full variant detail re-fetched and analyzed for property fingerprints. A "varying axis" is any property where ≥2 distinct non-empty values appear across the variants of a single chassis. "Property-duplicate" variants share an identical property fingerprint with another variant of the same chassis but have a different variant ID.

Sample: NZ = 1 week, UK = 3 days. Multi-variant rates are computed per market and combined. Per-brand sample sizes vary; small-sample brands (≤3 multi-variant VINs) shouldn't be over-interpreted. Brand attribution is only available for YES VINs — the NOs are excluded from per-brand frequency because their brand isn't returned by the API.
